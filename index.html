Fundamentos para construir un Sitio Web

Estos son algunos de los elementos claves para construir un sitio en HTML.

Etiquetas básicas

Para crear un título utilizamos la etiqueta <hn> donde n es cualquier número.
Por ejemplo:

<h1> Este es un título en html </h1>

Agregar imágenes

Para agregar una imagen debemos hacer uso de la etiqueta <img> y del atributo src. Por ejemplo:

<img src="> </img>
